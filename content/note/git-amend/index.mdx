---
title: '直前のコミットメッセージを修正する'
date: 2025-01-30

tags: ['git']
categories: [Git]
---

Git で直前のコミットメッセージを修正したい場合は `git commit --amend{:shell}` を使う。コマンド実行後にエディタが開くので、既にあるメッセージを修正し保存すればよい。

```shell
git commit --amend
```

`-m` オプションを追加すると、エディタを表示せずにコマンドラインから新しいメッセージを渡すこともできる。

```shell
git commit --amend -m 'commit message'
```
