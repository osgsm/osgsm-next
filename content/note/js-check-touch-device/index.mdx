JavaScript でタッチデバイスかどうかを判定する方法のひとつとして `Window.matchMedia(){:js}` を使う方法がある。

タッチデバイスは `'(hover: none)'` に該当するので、 `MediaQueryList.matches{:js}` プロパティを条件にし、それにマッチする場合の処理を書く。

```ts
if (window.matchMedia('(hover: none)').matches) {
  console.log('this is a touch device')
} else {
  console.log('this is a mouse device')
}
```
