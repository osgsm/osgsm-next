---
title: 'npm のキャッシュを完全に削除する'
date: 2025-04-09

tags: ['npm', 'cache', 'clean', 'clear']
categories: [npm]
---

npm のキャッシュを削除するには `npm cache clean` コマンドに `--force` オプションを渡す。

```shell
npm cache clean --force
```
