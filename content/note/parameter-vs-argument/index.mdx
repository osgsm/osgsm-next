---
title: 'Parameter と argument の違い'
date: 2025-02-10

tags: ['js']
category: JavaScript
---

Parameters は関数定義に含まれる変数であり、 arguments は、関数呼び出し時に渡す値のこと。

日本語では、前者は仮引数、後者は実引数と呼ばれる。

```js
const myFunction = (parameter) => {
  console.log(parameter)
}

const argument = 'foo'
myFunction(argument) // foo
```
