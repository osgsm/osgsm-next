---
title: 'pnpm で特定のバージョンにパッケージをアップグレードする'
date: 2025-01-27

tags: ['pnpm']
category: pnpm
---

pnpm ではパッケージのアップグレードを行うには `pnpm update` を使う。エイリアスとして `up` と `upgrade` が使える。

`@` を使えば特定のバージョンにアップグレードできる。

```shell
pnpm up shiki@1.29.1
```

ちなみに引数を渡さない場合は、すべてのパッケージをアップグレードする。
