## プロジェクト概要

このプロジェクトは Next.js 16 を使用した個人ブログ/ポートフォリオサイトです。

**主な技術スタック:**

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- MDX (next-mdx-remote)
- Three.js (WebGPU / TSL)
- React Three Fiber (R3F)
- Leva（リアルタイムパラメータ UI）
